La premisa de esta aplicación es la creación de una herramienta para consultar 
el clima en diferentes ciudades. Está diseñada para permitir al usuario seleccionar una 
ciudad desde un menú desplegable o ingresando el nombre de la ciudad en un campo de 
entrada de texto. Al presionar un botón, la aplicación utiliza la API de OpenWeatherMap
para obtener datos meteorológicos como la temperatura actual, la descripción del clima,
la sensación térmica, la humedad, la presión atmosférica y la velocidad del viento para
la ciudad seleccionada. 🌍

El diseño de la aplicación está orientado a ser amigable y fácil de usar, con una interfaz 
centrada y colores agradables. Se han aplicado estilos responsivos para adaptarse a diferentes
tamaños de pantalla, asegurando que la experiencia sea consistente en dispositivos móviles
y de escritorio. 🎨

El código utiliza principalmente HTML, CSS y JavaScript puro para la estructura, diseño
y funcionalidad respectivamente. Se ha incluido un ícono indicativo que destaca que 
la aplicación ha sido creada con JavaScript puro. 💻
